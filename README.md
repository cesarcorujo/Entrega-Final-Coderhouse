# Simulador de Reservas de Hoteles

## Descripción

Este proyecto fue desarrollado como entrega final del curso de JavaScript de Coderhouse.

La aplicación simula una página de reservas de hoteles, donde el usuario puede visualizar hoteles disponibles, buscar por nombre, filtrar por ciudad, ordenar por precio y realizar una reserva.

---

## Funcionalidades

- Mostrar hoteles cargados desde un archivo JSON mediante Fetch.
- Buscar hoteles por nombre.
- Filtrar hoteles por ciudad.
- Ordenar hoteles por precio.
- Agregar hoteles a una reserva.
- Calcular el total de la reserva.
- Guardar la reserva utilizando LocalStorage.
- Confirmar la reserva mediante SweetAlert2.

---

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Fetch API
- LocalStorage
- SweetAlert2
